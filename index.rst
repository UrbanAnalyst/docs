Urban Analyst
#############

This is the documentation of `the Urban Analyst platform
<https://urbananalyst.city>`_. Contributions to, or questions regarding, this
documentation, are welcome at `this GitHub repository
<https://github.com/UrbanAnalyst/docs>`_.

Following the introduction, the :ref:`second chapter<Overview>` provides a
brief overview of the main components of the Urban Analyst platform. This is
followed by a :ref:`walk-through example<Example>` comparison between Berlin,
Germany and Paris, France, illustrating the kinds of comparisons enabled by the
Urban Analyst platform. The |varchapter| describes all variables included in
the *Urban Analyst* platform, and the final two chapters describe all
|datachapter| used to derive these variables, and descriptions of all
|swchapter| used to generate the UA variables, including links to all
open-source software repositories.
